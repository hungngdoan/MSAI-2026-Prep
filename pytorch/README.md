# PyTorch Fundamentals (learnpytorch.io)

Chapters 00-09 from [learnpytorch.io](https://www.learnpytorch.io/).

## Structure

Each chapter has two notebooks:
- **sample/** - Follow along with learnpytorch.io, read and run code
- **exercise/** - Type code line by line from memory to build muscle memory

## Chapters

| Folder | Topic |
|--------|-------|
| `ch00_tensors` | Tensor creation, dtypes, shapes, devices, operations, reshaping |
| `ch01_workflow` | Synthetic data, train/test split, nn.Module, loss, optimizer, training loop |
| `ch02_classification` | Binary classification, BCEWithLogitsLoss, sigmoid, ReLU |
| `ch03_cnn` | FashionMNIST, DataLoader, CNNs, TinyVGG |
| `ch04_custom_data` | ImageFolder, transforms (Resize, ToTensor, Normalize) |
| `ch05_modular` | Convert notebooks to .py scripts, CLI training |
| `ch06_transfer_learning` | Pretrained models (EfficientNet_B0), freeze layers, feature extraction |
| `ch07_experiment_tracking` | Track and compare experiments, TensorBoard, monitoring training runs |
| `ch08_paper_replicating` | Replicate a ML research paper, implement architecture from scratch |
| `ch09_model_deployment` | Deploy model to the internet, export models, build a demo app |

## Workflow

1. Open **sample** notebook in one window
2. Open **exercise** notebook in another window
3. Work through the sample, then practice typing the code from memory in the exercise
